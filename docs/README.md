## Getting Started

### Web Application
Deploy the full QA Studio platform to AWS. See [Web App README](../web-app/README.md) for deployment instructions.

### CLI Tool
Install the command-line tool for local development and test execution. See [CLI README](../qa-studio-cli/README.md) for installation and usage.

### Agent Skill
Install the QA Studio agent skill for Kiro IDE and other coding agents using `npx skills add`. See [CLI README](../qa-studio-cli/README.md#kiro-ide-skill-installation) for setup instructions.
